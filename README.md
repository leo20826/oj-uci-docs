# K23OJ Documentation

Documentation for the [K23OJ](https://github.com/huythedev/K23OJ) system.

K23OJ is based on [VNOJ](https://github.com/VNOI-Admin/OJ), which itself is built on top of the [DMOJ](https://github.com/DMOJ/online-judge) 

See it live at **https://k23oj.io.vn/**

Access the documentation at <https://docs.k23oj.io.vn>.
